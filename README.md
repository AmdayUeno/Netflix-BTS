💜 Netflix BTS Experience

Uma interface inspirada na Netflix, recriada com uma identidade visual baseada na banda BTS e no conceito do álbum "Arirang".

Este projeto foi desenvolvido durante a Imersão de Front-end com IA da Alura, explorando não só o desenvolvimento web, mas também criatividade, design e o uso de inteligência artificial como aliada no processo de construção.

---

✨ Ideia do Projeto

A proposta foi ir além de um simples clone.

A ideia foi combinar:

- A estrutura e experiência da Netflix
- Uma identidade visual autoral
- Referências culturais e artísticas (BTS)

Criando assim uma experiência visual única.

---

🧠 IA no Desenvolvimento

Durante o projeto, utilizei inteligência artificial como suporte para:

- Construção e organização do código
- Ajustes de layout
- Sugestões de melhoria
- Correção de erros

---

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript
- Inkscape (design e redesenho das logos)

---

## 🎨 Identidade Visual

Toda a parte visual foi pensada de forma autoral:

- Inspiração no BTS
- Redesenho da logo do projeto
- Elementos visuais personalizados
- Adaptação do estilo da Netflix para um novo contexto

---

## 📱 Experiência

O projeto foi desenvolvido com foco em:

- Layout responsivo
- Interações suaves
- Organização visual
- Experiência do usuário
